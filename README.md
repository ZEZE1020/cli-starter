# CLI Starter Course

A beginner-friendly, step-by-step CLI course for **Mac** and **Windows (Git Bash)** users.

📖 **[Read the full course on GitHub Pages →](https://zeze1020.github.io/cli-starter)**

---

## Course Contents

| Lesson | Topic |
|--------|-------|
| 1 | [CLI Basics](docs/01-cli-basics.md) — open a terminal, first commands, keyboard shortcuts |
| 2 | [Navigation & File Operations](docs/02-navigation-and-files.md) — `ls`, `cd`, `mkdir`, `cp`, `mv`, `rm` |
| 3 | [Bootstrap a Project](docs/03-bootstrap-project.md) — `git init`, `npm init`, first commit |
| 4 | [Create Your Portfolio](docs/04-create-portfolio.md) — build and deploy to GitHub Pages |

## Prerequisites

- **Mac** — Terminal (built-in)
- **Windows** — [Git Bash](https://gitforwindows.org/)
- **Node.js / npm** — [nodejs.org](https://nodejs.org) (LTS)
- **GitHub account** — [github.com](https://github.com)

## Local Development

```bash
gem install bundler
bundle install
bundle exec jekyll serve
# Open http://localhost:4000/cli-starter
```
